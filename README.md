# 회고 프로젝트

## 오늘의 할 일 협업 대시보드

### 기능
1. 회원 관리 및 보안
    1. 회원 가입 / 로그인
        - `BCrypt` 암호화를 적용한 계정 생성 및  세션 기반의 상태 관리
        - 인가 : 모든 API 호출 시 세션으로  "로그인한 사용자"인지 확인

    3. 회원 탈퇴
        - 
2. 대시보드 / 보드 관리
    1. 보드 생성 
        - 보드 이름과 설명 입력하여 생성
    2. 나의 보드 목록 조회
        - 내가 생성했거나 초대받은 보드들만 필터링하여 조회
    3. 보드 수정 / 삭제
        - 보드장만 보드의 설정이나 이름을 바꿀 수 있도록 JWT 기반 권한 체크
3. 할 일 카드 관리
    1. 카드 생성
        - 특정 보드 내에 할 일(제목, 내용, 마감일 등) 추가
    2. 카드 상태 변경
        - 할일 → 진행 중 → 완료
    3. 담당자 지정
        - 해당 보드에 속한 멤버 중 한 명을 카드 담당자로 지정
    4. 카드 수정 / 삭제
        - 카드 생성자 or 보드 참여자만 수정 가능
4. 협업 및 멤버 초대
    1. 멤버 초대
        - 보드장이 이메일로 다른 회원을 보드에 초대
    2. 참여자 권한 체크
        - 멤버가 아닌 사용자가 API 통해 카드 조회하려 하면 403 에러 반환
